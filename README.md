# simple-gorilla_mux.go

「gorilla/mux」に関する学習目的プログラム。  
[gorilla](https://github.com/gorilla/mux)  

## 実行方法

```shell
# デバグ実行
go run main.go

# ビルド
go build -o bin main.go
```

## メモ

プログラムファイルにてあるパッケージをインポートし、「go mod tidy」を実行すると自動でダウンロードしてくれる!!!
